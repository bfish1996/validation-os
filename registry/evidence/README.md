# Raw evidence — local-only

Raw evidence material lives here: interview transcripts, chat exports,
recordings, and the identities behind anonymized register records.

Everything in this directory except this README is **gitignored**. Register
records are also local-only by default, with a few published as worked
examples (`registry/README.md`) — published records stay real but anonymized,
referencing files here by local path ("Founder A", source:
`registry/evidence/<date>-<slug>.md`) so the method is auditable in public
without publishing anyone's private messages or name.

Convention:

- One file per piece of raw evidence: `YYYY-MM-DD-<slug>.md`.
- Top of file: who the anonymized label ("Founder A") actually is, and the
  consent/context of the material.
- Below: the raw material, verbatim.
- Public register rows quote sparingly and generically; the full text stays
  here.
